# Meta Multimodal Retrieval
> [!CAUTION]
> **Project Status: In Progress**


This project demonstrates **multimodal retrieval** across video, image and text
using **Meta Research Perception Models** and **SAM-3 segmentation**.

## Features
- Text → Video/Image retrieval
- Video → Text retrieval
- Image → Image similarity
- FAISS-based similarity search
- Streamlit interactive demo
- Minimal, production-like, Meta-zuordenbar

## Architecture
- meta_models as Git Submodule
- SAM-3 integration for mask embeddings
- Embedding pipeline (video frames, images, text)
- FAISS for efficient retrieval
- Streamlit for demo
